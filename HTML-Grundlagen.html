HTML-Grundlagen

Mittels HTML (Hypertext Markup Language) definieren wir die Struktur einer HTML-Seite, die Tags und normalen Text enthalten kann. Tags setzen wir in eckigen Klammern, wobei Start- und Endtags unterschieden werden. Bei einer Webseite kannst Du Dir mit Strg + U den Quelltext einer HTML-Seite ansehen.

Der Browser, der das HTML interpretiert, bringt zur Darstellung eigene CSS3-Anweisungen mit, die das Erscheinungsbild der verschiedenen Elemente im Browser festlegen.

Überschriften

Mit <h1></h1>, <h2></h2>, <h3></h3>, <h4></h4>, <h5></h5> und <h6></h6> können wir Überschriften definieren. h steht dabei für header.

      <h1>Die erste Überschrift</h1>
      <h2>Die zweite Überschrift</h2>
      <h3>Die dritte Überschrift</h3>
      <h4>Die vierte Überschrift</h4>>
      <h5>Die fünfte Überschrift</h5>
      <h6>Die sechste Überschrift</h6>

Absätze

Für Absätze wird das <p></p>-Tag (paragraph) verwendet. Ohne das Element würde Text einfach aneinander gehängt. Das <p></p>-Element bewirkt eine sinnvolle Gliederung nach Absätzen.

      <p>Der erste Absatz</p>
      <p>Der zweite Absatz</p>

Möchte man in einem Absatz einen Zeilenumbruch erzwingen, dann verwendet man dafür das <br>-Tag, das kein schließendes Element besitzt.

Nachfolgend zwei Beispiele, einmal ohne und einmal mit <p>-Tag:

      Der erste Absatz.
      Der zweite Absatz.

Ergibt dann im Browser:

Der erste Absatz. Der zweite Absatz.

Mit dem Tag sieht es dann wie folgt aus:

      <p>Der erste Absatz.</p>
      <p>Der zweite Absatz.</p>

Im Browser sieht man dann:

Der erste Absatz.

Der zweite Absatz.

Listen

HTML kennt 2 Möglichkeiten, Listen hinzuzufügen und zwar einmal normale Listen:

      <ul>
          <li>Listenelement 1</li>
          <li>Listenelement 2</li>
      </ul>

So sieht es dann auf der Seite aus:

  • Listenelement 1
  • Listenelement 2

Ferner gibt es noch eine nummerierte Liste:

      <ol>
          <li>Listenelement 1</li>
          <li>Listenelement 2</li>
      </ol>

So sieht es dann auf der Seite aus:

  1. Listenelement 1
  2. Listenelement 2

Bilder einfügen

Bilddateien werden mit dem <img>-Tag in einer HTML-Seite eingebunden. Ein Beispiel:

      <img src="path/filename.png" alt="Beschreibung des Bildes" width="200px" height="400px" />

In diesem Beispiel wird eine Bilddatei aus dem Ordner path mit dem Dateinamen filename.png geladen und mit der Breite 200px und der Höhe 400px in der Seite angezeigt. Mit dem Element alt kann man das Bild beschreiben, dieser Text wird angezeigt, wenn das Bild nicht geladen werden konnte.

In der Regel beschränkt man sich darauf, entweder die Höhe oder die Breite anzugeben, der andere Wert wird dann jeweils berechnet. Beachte, dass das img-Tag kein schließendes Tag besitzt.

Bilder aus Verzeichnissen laden

Grundsätzlich ist es möglich, ein Bild aus demselben Ordner zu laden, in dem auch die HTML-Seite liegt, man kann aber ein Bild auch aus einem Unterordner bzw. aus einem Ordner laden, der oberhalb des Verzeichnisses mit der HTML-Datei liegt. Auch das Laden einer Bilddatei direkt aus dem Netz ist möglich.

Wir nehmen folgende Verzeichnisstruktur an:

  • Ordner 1
    • Datei_1.png
  • Ordner 2
    • Datei_2.png
    • index.html
      • Ordner 3
        • Datei_3.png

Liegt die Bilddatei im selben Verzeichnis wie die HTML-Datei, dann kann man die Bilddatei wie folgt laden:

            <img src="Datei_2.png" alt="Beschreibung des Bildes" width="200px" height="400px" />

Liegt die Bilddatei in einem anderen Verzeichnis, dann kann man mittels .. eine Ebene höher wechseln und von dort in das Verzeichnis mit der Bilddatei wechseln. Die Zeile mit dem img-Tag lautet dann:

            <img src="../Ordner 1/Datei_1.png" alt="Beschreibung des Bildes" width="200px" height="400px" />

Liegt die Bilddatei in einem Verzeichnis unterhalb der index.html-Datei, dann lautet die Zeile mit dem img-Tag

            <img src="Ordner 3/Datei_1.png" alt="Beschreibung des Bildes" width="200px" height="400px" />

Links

Links auf andere Seiten

Für die Navigation zu anderen Seiten nutzt man Hyperlinks, die wie folgt aufgebaut werden:

        <a href="index.html">Hauptseite</a>

Im Browser angezeigt wird dann:

Hauptseite

Für die Pfadangaben, d. h. die Navigation innerhalb von Verzeichnissen gelten die selben Hinweise wie bei Bildern.

E-Mail-Adressen verlinken

E-Mail-Adressen kann man wie folgt in einer HTML-Seite verlinken:

        <a href="mailto:dummy@www.de">Kontakt</a>

Im Browser angezeigt wird dann:

Kontakt

Wird ein mailto:-Link angeklickt, wird beim Benutzer versucht, für den Link ein Mailprogramm zu öffnen.

Tabellen

Tabellen können in HTML-Seiten genutzt werden, z. B. zur Darstellung tabellarischer Daten. Im HTML-Quelltext wird eine Tabelle wie folgt definiert:

      <table>
          <thead>
              <tr>
                  <th>Spalte 1</th>
                  <th>Spalte 2</th>
              </tr>
          </thead>
          <tbody>
              <tr>
                  <td>Zelle 1</td>
                  <td>Zelle 2</td>
              </tr>
              <tr>
                  <td>Zelle 3</td>
                  <td>Zelle 4</td>
              </tr>
          </tbody>
      </table>

Im Browser wird dann folgendes angezeigt:

Spalte 1 Spalte 2
Zelle 1 Zelle 2
Zelle 3 Zelle 4

Beachte, dass die Rahmen der Tabelle standardmäßig nicht angezeigt werden.

div-Boxen

HTML bietet die Möglichkeit größere Bereiche einer Seite mit einer div-Box zu umgeben und so die Seite zu strukturieren. div-Boxen spielen eine große Rolle für CSS-Layouts, da sie unterschiedlich formatiert werden können. Standardmäßig nehmen sie die volle Breite einer HTML-Seite ein.

Wir definieren eine div-Box wie folgt:

<div>Hier ist Platz für einen Inhalt.</div>

Im Browser ergibt sich dann:

Hier ist Platz für einen Inhalt.

Hervorhebung einzelner Wörter oder Satzteile

Mit den Tags <strong></strong> (Fettdruck), <em></em> (Kursivdruck) und <u></u> (Unterstreichung) kann man Text hervorheben. Da Hyperlinks standardmäßig unterstrichen werden, sollte man, um Verwechslungen zu vermeiden, auf Unterstreichungen möglichst verzichten.

Wichtig in diesem Zusammenhang ist auch das span-Element, mit dem Wörter oder längere Textabschnitte markiert werden können. Wichtig ist das vor allem, weil diese Elemente dann eine ID (id) oder eine Klasse (class) erhalten können und damit wiederum für CSS-Formatierungen ansprechbar werden. Ohne eine CSS-Formatierung ist ein span-Element auf der HTML-Seite nicht erkennbar.

Wir nutzen das span-Element wie folgt:

      <p>Dieser Satz stellt <span>nur</span> ein Beispiel dar.</p>

Im Browser ist die Auszeichnung so noch nicht zu sehen:

Dieser Satz stellt nur ein Beispiel dar.

Weitere HTML-Tags

Kommentare kann man zwischen den Elementen <!-- und --> einfügen. Die HTML-Kommentare werden im Browser nicht angezeigt, bleiben aber im Quelltext einer HTML-Seite erhalten.

Mit dem <hr>-Tag kann man eine horizontale Linie auf einer Seite einfügen. Das Element hat kein schließendes Element.

Der Browser ordnet die Zeilen nach eigenem Gutdünken an. Möchte man verhindern, dass ein Text an einem Leerzeichen in einer Zeile umgebrochen wird, kann man ein sogenanntes geschütztes Leerzeichen verwenden und ersetzt hierzu ein Leerzeichen durch  .
